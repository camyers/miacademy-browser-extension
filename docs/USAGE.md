# Usage Guide

## Basic Usage

1. **Navigate to Miacademy.co** and log in to your account
2. **Click the extension icon** in your browser toolbar to open the popup
3. **Toggle checkboxes** for the elements you want to hide
4. **Changes apply immediately** - no need to reload the page
5. **Use "Select All" or "Deselect All"** for quick bulk changes

## Saving Preferences

Your preferences are automatically saved and will persist across:
- Page reloads
- Browser sessions
- Different devices (if you're signed into your browser account)

## Applying Changes

- **Individual Checkboxes**: Changes apply immediately when toggled
- **Select All / Deselect All**: Bulk changes apply immediately
- **Apply Changes Button**: Applies changes and closes the popup
- **Cancel Button**: Closes the popup without additional actions
- **X Icon**: Closes the popup from the header

## Tips

- Changes are saved automatically - no need to click "Apply Changes" unless you want to close the popup
- Use "Select All" at the top to quickly enable all hiding options
- Preferences sync across devices if you're signed into your browser account
