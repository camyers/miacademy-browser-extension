A browser extension that allows you to customize your Miacademy.co experience by hiding specific UI elements based on your preferences. Perfect for creating a cleaner, more focused learning environment.

## Quick Start

1. **Install** the extension (see [Installation Guide](docs/INSTALLATION.md))
2. **Navigate** to Miacademy.co and log in
3. **Click** the extension icon to open the popup
4. **Toggle** checkboxes to hide elements
5. **Changes apply immediately** - no reload needed!

## Features

- Hide navigation, sidebar, header, footer, and global elements
- Select All / Deselect All for bulk changes
- Persistent preferences synced across devices
- Real-time updates without page reload
- Domain-specific (only works on miacademy.co)

## Installation

Quick installation steps:

**Edge/Chrome:**
1. Go to `edge://extensions/` or `chrome://extensions/`
2. Enable Developer mode
3. Click "Load unpacked"
4. Select this folder

**Firefox:**
1. Go to `about:debugging`
2. Click "Load Temporary Add-on"
3. Select `manifest.json`
